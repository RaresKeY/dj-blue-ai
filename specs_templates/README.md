# Specs Template Kit

Use these templates to bootstrap a spec system in a new project.

## Quick Start
1. Copy `specs_templates/_readme.template.md` to `<new-project>/specs/_readme.md`.
2. Keep templates under `<new-project>/specs_templates/` to preserve separation from ground-truth specs.
3. Copy selected `*.template.md` files from `specs_templates/` into `<new-project>/specs/` and remove `.template` from filenames.
4. Replace placeholders (`<...>`) with concrete project details.
5. Keep spec docs updated whenever architecture, behavior, or constraints change.

## Included Templates
- `_readme.template.md`
- `product_context.template.md`
- `system_architecture.template.md`
