# Gallery Layout Map

This document maps the 2-column grid layout generated by `snap_gallery.py` to its source previews, views, and related components.

## Grid Arrangement

The gallery is a 2-column vertical strip. Each "cell" is a high-resolution snapshot of a specific window or state.

| Row | Column 0 (Left) | Column 1 (Right) |
|:---:|:---|:---|
| **0** | **Main Window** | **Transcript Window** |
| **1** | **BlueBird Chat** | **User Profile** |
| **2** | **Settings: Recording** | **Settings: Theme** |
| **3** | **Settings: Music** | **Settings: Limits** |

---

## Component Origins

### Core Windows (Rows 0-1)

| Name | Preview Script | Primary View Class | Related Helpers/Widgets |
|:---|:---|:---|:---|
| Main Window | `preview_main_window.py` | `MainWindowView` | `song_cover_carousel.py`, `timeline.py`, `volume.py` |
| Transcript | `preview_transcript_window.py` | `TranscriptWindowView` | `text_boxes.py` (TextBox, SearchBar) |
| BlueBird Chat | `preview_chat_window.py` | `BlueBirdChatView` | `gemini_chatbot.py`, `text_boxes.py` (TextBoxAI) |
| User Profile | `preview_profile_window.py` | `ProfileWindowView` | `resource_path.py` (Avatar asset) |

### Settings Tabs (Rows 2-3)

| Tab Name | Preview Class | View Class | Related Logic/Widgets |
|:---|:---|:---|:---|
| Recording Sources | `SettingsPopupPreview` | `SettingsPopup` | `tabs_audio.py` (Source enumeration) |
| Theme Selection | `SettingsPopupPreview` | `SettingsPopup` | `theme_chooser.py`, `manager.py` |
| Music Library | `SettingsPopupPreview` | `SettingsPopup` | `config/__init__.py` (Music folder logic) |
| API Usage Limits | `SettingsPopupPreview` | `SettingsPopup` | `api_usage_limits_form.py`, `api_usage_guard.py` |

## Tooling Context
- **Orchestrator**: `ui_ux_team/blue_ui/previews/snap_gallery.py`
- **Capture Engine**: `ui_ux_team/blue_ui/previews/ui_iterate.py` (snap command)
- **Theme Logic**: Uses `DJ_BLUE_THEME_OVERRIDE` to switch themes headlessly without persistent config changes.
